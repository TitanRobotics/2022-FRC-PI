 ----------------------------------------------------------------
/  1. Using Marvin in your a application                        /
----------------------------------------------------------------

- copy this folder 'marvin' to your project folder and add marvin_1.4.2.jar into your project 
  library list
- the folder organization, including subfolders 'framework' and 'plugins', must be preserved.
- DO NOT add plug-ins .jar to your library list. They are loaded dynamically.

 ----------------------------------------------------------------
/  2. Compiling and running (command-line)                      /
----------------------------------------------------------------

Compiling: javac -cp .;./marvin/framework/marvin_1.4.2.jar YourApplication.java

Running: java -cp .;./marvin/framework/marvin_1.4.2.jar YourApplication
